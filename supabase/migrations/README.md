# מיגרציות מסד נתונים (Database Migrations)

**זו התיקייה היחידה והמיועדת לכל מיגרציות ה-DB בפרויקט.**

- כל קבצי המיגרציה (`.sql`) הקיימים והעתידיים נמצאים כאן.
- שמות הקבצים: `NNN_תיאור_קצר.sql` (למשל `031_xxx.sql`).
- הרצה: `supabase db push` או הרצת ה-SQL ידנית ב-Supabase Dashboard.

**אין ליצור מיגרציות בתיקיות אחרות.**
